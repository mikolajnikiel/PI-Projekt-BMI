BMI to wskaźnik pokazujący stosunek masy ciała do wzrostu.
BMI posiada róźne klasyfikacje ciał, wskazują one kolejno: niedowage, wage optymalną, nadwagę i otylość, każda z klasyfikacji określana jest przez BMI, niedowaga - najniższy wskaźnik, otyłość - najwyższy.
